## Featured Projects

### Winery Production ETL Pipeline
*Python · PostgreSQL · Pandas · psycopg2 · Star Schema*

Designed and built a production-grade ETL pipeline starting from a 300,000-row wine sales dataset with significant temporal gaps. Used Python and Faker to synthesize realistic operational records, scaling the final dataset to **435,000+ rows**. Transformed a flat CSV into a normalized **star schema** in PostgreSQL — with dimension tables for suppliers, item types, products, and markets — enabling complex window functions, inventory analytics, and distribution trend queries across SKUs, regions, and time periods.

### ML Wine Quality Predictor
*Python · Scikit-learn · XGBoost · Streamlit · Pandas*

End-to-end ML classification pipeline predicting wine quality scores from physicochemical properties (acidity, sulfates, alcohol, etc.). Trained and benchmarked four models — Logistic Regression, Random Forest, SVM, and XGBoost — comparing accuracy, F1, precision, recall, and ROC-AUC. Deployed as an interactive Streamlit app with live prediction, feature importance visualization, and model comparison pages.

### Central Valley Crop Economics Dashboard
*R · flexdashboard · Plotly · USDA NASS API · Tidyverse*

Interactive R Markdown dashboard analyzing agricultural economics across **7 Central Valley counties** (Fresno, Tulare, Kings, Merced, San Joaquin, Stanislaus, Madera) for 6 major crops — almonds, grapes, walnuts, processing tomatoes, dairy, and cotton — from 2010 to 2025. Pulls live data from the **USDA NASS Quick Stats API**, handles a two-tier county/state data architecture, and delivers six dashboard pages: value of production trends, price received over time, county comparisons, yield index (2015=100), and a COVID & drought impact analysis. Directly relevant to California's $50B+ agricultural economy.

[View Project →](https://github.com/jorgecreyesm/Portfolio/tree/main/R_Projects/Central_Valley_Ag_Economics)
