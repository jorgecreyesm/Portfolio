# ☀️ Solar Paradox: Global Solar Energy Adoption Analysis

## 📁 Project Structure

This folder contains both the **interactive Streamlit app** and the **original research files**:

### 🌐 Interactive App (Primary)
- **`app_solar.py`** - Production Streamlit application with 7 interactive pages
- **`requirements.txt`** - Python dependencies
- **`.streamlit/config.toml`** - App styling and configuration
- **`STREAMLIT_README.md`** - App usage guide
- **`solar_energy_worldwide.csv`** - Dataset (committed to Git for deployment)

### 📊 Original Research Files (Reference)
- **`solar_paradox.py`** - Original analysis script with 5 plot functions
  - Used to generate the initial analysis and insights
  - Kept for reproducibility and reference implementation
  - Shows the analytical approach before Streamlit conversion
  
- **`SolarSustainability.ipynb`** - Original Jupyter notebook
  - Contains exploratory data analysis and methodology
  - Documents the research process and findings
  - Useful for understanding the analytical decisions

## 🚀 Running the App

```bash
pip install -r requirements.txt
streamlit run app_solar.py
```

## 📖 Why Both Files Exist

1. **Transparency** - Shows the research-to-production pipeline
2. **Reproducibility** - Original scripts document analytical methodology
3. **Portfolio Value** - Demonstrates full analytics workflow (analysis → visualization → deployment)
4. **Documentation** - Notebooks serve as extended documentation
